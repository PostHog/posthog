# phdev

PostHog-branded dev process runner built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).
Drop-in replacement for `mprocs` вҖ” reads the same YAML config that `hogli dev:generate` produces.

## Usage

```sh
phdev <config.yaml>
phdev --debug <config.yaml>   # write debug log to /tmp/phdev-debug.log
```

The config is generated by `hogli dev:generate` and lives at `.cache/hogli/dev.yaml` in the repo root.
You typically run phdev via `hogli dev` rather than directly.

## Keybindings

| Key                 | Action                   |
| ------------------- | ------------------------ |
| `j` / `tab`         | Next process             |
| `k` / `shift+tab`   | Previous process         |
| `вҶ‘` / `вҶ“`           | Scroll output            |
| `ctrl+u` / `ctrl+d` | Scroll half page         |
| `g` / `G`           | Jump to top / bottom     |
| `r`                 | Restart selected process |
| `?`                 | Toggle full help         |
| `q` / `ctrl+c`      | Quit                     |

Mouse scroll is also supported in the output pane.

## Debug logging

Pass `--debug` to write a timestamped log of TUI events to `/tmp/phdev-debug.log`:

```log
2026/03/06 10:00:00.123456 debug logging started
2026/03/06 10:00:00.145678 resize: 220x54
2026/03/06 10:00:00.200000 status: proc=backend status=running
2026/03/06 10:00:01.300000 key: "j" (type=runes)
2026/03/06 10:00:01.300100 proc selected: 0вҶ’1 (frontend)
```

Tail it in a separate terminal while phdev is running:

```sh
tail -f /tmp/phdev-debug.log
```
