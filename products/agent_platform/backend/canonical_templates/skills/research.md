# Research

Use this skill when the user asks an open-ended question that requires
combining results from multiple tools or sources before answering.

## When to use

- The question is exploratory: "what happened around X?", "why is Y trending?"
- The answer needs at least two data points pulled from distinct surfaces.
- You don't already know the answer with high confidence.

## Process

1. **Frame the question.** Restate what's being asked in one sentence to
   confirm your understanding.
2. **Pick the right surfaces.** Identify which tools you'll need (events,
   logs, documentation, etc.) before calling any of them.
3. **Pull data incrementally.** Make one call, read the result, decide
   whether you need another.
4. **Synthesize.** Reply with what you found, where you found it, and
   what you're not certain about.

## What to avoid

- Don't run every tool at once "to be safe" — pick the smallest set that
  could answer the question.
- Don't fabricate confidence. If two sources disagree, say so.
